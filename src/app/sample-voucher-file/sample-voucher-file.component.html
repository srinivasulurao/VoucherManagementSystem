Bulk Voucher Uploading File Details

Expected CSV File Format.
Rows/ColumnsDetails
ROW 1This row should be untouched, this is the expected format using which the system will process the voucher/coupon uploading task.
coupon_code * In this column you should add all the coupon code which your site customer will use it for redemption. Data for this column is mandatory.
voucher_validity * This column you should include the voucher expiration date, the voucher/coupon will stop working after this mentioned date. Data for this column is mandatory.
productIn this column you can mention the product to which the voucher/coupon should be linked. The product name should be same as that of present in the system. Data for this column is optional.
notesHere you can some notes related to the voucher. Data for this column is optional.
