Sample Voucher File Details

Expected Format
Rows/ColumnsDetails
ROW 1This row should be untouched, this is the expected format using which the system will process the uploading task.
Column 1 - Required (coupon_code)In this column, you should add all the coupon code which your site customer will use it for redemption.
Column 2 - Required (voucher_validity)This column should include the voucher expiration date, the voucher/coupon will stop working after this mentioned date.
Column 3 - Optional (product)In this column you should mention the product to which the voucher/coupon should be linked. The product name should be same as that of present in the system.
Column 4 - Optional (notes)Here you can some notes related to the voucher.
